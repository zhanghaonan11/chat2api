# chat2api 本机部署命令

项目路径：`/Users/shan/github/2api/chat2api`

LaunchAgent：`com.shan.chat2api`

配置文件：`/Users/shan/github/2api/chat2api/conf/app.dev.yaml`

本机地址：`http://127.0.0.1:3040`

局域网地址示例：`http://192.168.1.125:3040` 或 `http://192.168.1.197:3040`

## 安装或重装后台服务

```bash
cd /Users/shan/github/2api/chat2api
chmod +x scripts/macos/*.sh
./scripts/macos/install_launchd.sh
```

## 查看状态

```bash
launchctl print gui/$(id -u)/com.shan.chat2api
lsof -nP -iTCP:3040 -sTCP:LISTEN
curl -sS --max-time 5 http://127.0.0.1:3040/ping
curl -sS --max-time 5 http://192.168.1.125:3040/ping
```

## 重启

```bash
launchctl kickstart -k gui/$(id -u)/com.shan.chat2api
```

## 停止并卸载

```bash
cd /Users/shan/github/2api/chat2api
./scripts/macos/uninstall_launchd.sh
```

## 日志

```bash
tail -f /Users/shan/github/2api/chat2api/logs/launchd.stdout.log
tail -f /Users/shan/github/2api/chat2api/logs/launchd.stderr.log
tail -f /Users/shan/github/2api/chat2api/logs/app.dev.log
```

## API 验证

```bash
curl -sS http://127.0.0.1:3040/ 
curl -sS http://127.0.0.1:3040/ping
curl -sS http://127.0.0.1:3040/v1/accTokens \
  -H 'Authorization: Bearer sk-local-809f8cff083258e057cfd3cd28c965294d2444cc481eb7cf'
```

`chatgpts` 当前为空，只能验证服务和本地鉴权链路；要实际请求 ChatGPT 上游，需要把真实 `access_token` 填进 `conf/app.dev.yaml` 的 `chatgpts`。

示例：

```yaml
chatgpts:
  - access_token: your_chatgpt_access_token_here
    type: chatgpt
    proxy: ""
```
